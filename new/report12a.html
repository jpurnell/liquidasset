 We now compute two correlations for each taster:(1)  R, which is the correla-
 tion of a taster's ranking with the average ranking of the others, (2) I,
 which is the correlation of each taster's wine identification with
 the TRUE  identification of the wines. R can have values from 1.0 (perfect
 correlation) to -1.0 (perfect disagreement) and I can have values from 1.0
 (perfect match with the truth) to 0 (no match)

Remarks:

We tested the same wines as the night before. Each bottle had about 20 hours time to breathe. We corrected the mistake with the Busch Spaetlese*** and inserted a Busch Auslese. It was also opened the night before. Everybody agreed that it is much easier to the strengths and weaknesses of the wines (although all of us had a hard night). Maybe this is the reason for the higher correlation among the judges than the night before. The theory was proposed that the correlation might have been much higher if the wines had been opened 3 or 4 days before the tasting (see the October 1998 tasting).
